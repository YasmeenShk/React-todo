# React Todo App

A clean and responsive **Todo List application built with React and Tailwind CSS**.
It helps users manage daily tasks with features like adding, editing, deleting, filtering, and dark mode support.

## 🌐 Live Demo

[React ToDo Live ](ys-react-todo.netlify.app)


## 🚀 Features

* ➕ Add new tasks
* ✏️ Edit tasks
* 🗑️ Delete tasks
* ✅ Mark tasks as completed
* 🔍 Filter tasks (All / Active / Completed)
* 🌙 Dark / Light mode toggle
* 💾 Tasks saved in Local Storage
* 📱 Responsive UI with Tailwind CSS

## 🛠️ Tech Stack

* React.js
* Tailwind CSS
* JavaScript (ES6)
* HTML5
* CSS3

## ⚙️ Installation

Clone the repository:

```
git clone https://github.com/YasmeenShk/React-todo.git
```

Navigate to the project folder:

```
cd React-todo
```

Install dependencies:

```
npm install
```

Run the project:

```
npm run dev
```
